Original graph. The original graph is typical of a default, Excel generated graph. There are several improvements we can make to the graph by simply removing clutter. The first item we'll remove is the border surrounding the entire graph. The border is unnecessary.

Border removed. Next we'll remove the horizontal gridlines running through the chart.

Horizontal gridlines removed. Next we'll remove the graph's data markers.

Data markers removed. Next we'll clean up the axis labels. In the horizontal axis, the months can be abbreviated to eliminate the need for the angled text. In the vertical axis, the two decimal point accuracy can be eliminated since it serves no purpose other than to add clutter.

Axis labels cleaned up. Next we'll label the graph's data directly in order to eliminate the need for the legend. The labels "Received" and "Processed" can be placed right next to their corresponding line graphs.

Data labeled directly. Lastly, we'll leverage consistent color. The colors in the data labels can be changed to match the colors in their corresponding line graphs.

Consistent color leveraged. Our finished graph improves viewer readibility. The viewer no longer has to wade through unnecessary clutter to perceive the story. To start over, click the next button again.
